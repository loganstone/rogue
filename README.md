# rogue
www.roguebasin.com 에 평소에 만들고 싶었던 로그라이크 게임의 튜토리얼이 있어 따라 해 보고, 개선하기

## Getting started

### Install

* Install python3 ;)

* Install `rogue`

```shell
$ git clone git@github.com:loganstone/rogue.git
$ cd rogue
$ pip install -r requirements.txt
```

* Run

```shell
$ python engine.py
```

- 간단한 로그라이크 게임.
